## Push ball - 2dgame

Esse game está em beta e incompleto, muitos _bugs_ serão encontrados.

é necessário ter o node instalado

após clonado:

#### to run in production mode:
```
$ npm install
$ npm run start

```
or

```
$ yarn
$ yarn start

```

#### to run in development mode:
```
$ npm install
$ npm run dev

```
or

```
$ yarn
$ yarn dev

```
